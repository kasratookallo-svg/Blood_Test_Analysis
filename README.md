# 🩸 Blood Test Analysis

A Python-based **Exploratory Data Analysis (EDA)** project for laboratory blood test data. This project imports blood test datasets, performs statistical and correlation analyses, and generates informative visualizations to explore relationships among clinical biomarkers.

---

## 📖 Overview

Blood test reports contain numerous biomarkers that can provide valuable insights into a patient's health. This project simplifies the exploration of laboratory data by applying statistical analysis and visualization techniques.

The project focuses on identifying relationships between blood parameters through correlation analysis and graphical representations, making it easier to detect patterns, trends, and potential associations within the dataset.

---

## ✨ Features

* Import blood test data from CSV files
* Data preprocessing and cleaning
* Descriptive statistical analysis
* Correlation matrix calculation
* Correlation heatmap visualization
* Pair plot generation
* Distribution analysis of laboratory parameters
* Easy-to-understand visual outputs

---

## 📊 Visualizations

The project generates several exploratory data analysis plots, including:

* Correlation Heatmap
* Pair Plot
* Histograms
* Distribution Plots
* Summary Statistics

These visualizations help identify:

* Relationships between biomarkers
* Highly correlated laboratory parameters
* Data distributions
* Potential outliers
* General trends within the dataset

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/kasratookallo-svg/Blood_Test_Analysis.git
```

Move into the project directory:

```bash
cd Blood_Test_Analysis
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the Jupyter Notebook or Python script to:

2. Perform exploratory data analysis.
